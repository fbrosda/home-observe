#+title: Home Observability

The project contains different components, to collect and monitor the following systems installed in my home:

- Kostal Plenticore PV Inverter
- IDM Heatpump
- Airvisio

There is a guile module, which connects to the different devices, via the available apis and collects the data, which is written into a timescaledb.

The grafana folder contains the configs for various dashboards generated from the collected data.

** Nginx config
#+begin_src nginx
# This is required to proxy Grafana Live WebSocket connections.
map $http_upgrade $connection_upgrade {
  default upgrade;
  '' close;
}

upstream grafana {
  server localhost:3000;
}

location /grafana/ {
    proxy_set_header Host $host;
    proxy_pass http://grafana;
}

location /grafana/api/live/ {
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection $connection_upgrade;
    proxy_set_header Host $host;
    proxy_pass http://grafana;
}
#+end_src
