#+title: Home Observability

The project collects and monitors data from the following home systems:

- Kostal Plenticore PV Inverter
- IDM Heatpump
- Airvisio (planned, [[https://www.airviso.de/wp-content/uploads/2025/07/airviso_ONE100_Bedienungs_und_Montageanleitung_09-2024.pdf][Bedienungs- und Montageanleitung]])

A Guile Scheme 3.0 daemon connects to each device via its API, collects metrics at 10-second intervals, and writes them to TimescaleDB.

The ~grafana/~ folder contains dashboard configurations generated from the collected data.

* Dependencies

- Guile 3.0
- Guile modules: ~dbi~, ~gcrypt~, ~guile-json~
- libgcrypt (for AES-GCM encryption)
- PostgreSQL / TimescaleDB

* Build & Install

#+begin_src sh
make install   # installs binary, modules, and systemd service
make uninstall # removes installed files
#+end_src

* Configuration

Config is read from ~/etc/home-observe.cfg~ as a Scheme s-expression:

#+begin_src scheme
((plenticore
  (host . "plenticore.fritz.box")      ;; optional, default shown
  (password . "your-password")
  (connection . "host=... dbname=... user=... password=..."))
 (idm
  (host . "idm.fritz.box")             ;; optional, default shown
  (password . "your-auth-code")
  (connection . "host=... dbname=... user=... password=...")))
#+end_src

The config file is gitignored as it contains secrets.

* TimescaleDB Setup

[[https://docs.timescale.com/self-hosted/latest/install/][Official Documentation]]

Tables are created automatically on first run via ~CREATE TABLE IF NOT EXISTS~ with TimescaleDB hypertable extension.

** Nginx config
#+begin_src nginx
# This is required to proxy Grafana Live WebSocket connections.
map $http_upgrade $connection_upgrade {
  default upgrade;
  '' close;
}

upstream grafana {
  server localhost:3000;
}

location /grafana/ {
    proxy_set_header Host $host;
    proxy_pass http://grafana;
}

location /grafana/api/live/ {
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection $connection_upgrade;
    proxy_set_header Host $host;
    proxy_pass http://grafana;
}
#+end_src
